RentEase 

RentEase is a simple Furniture and Appliance Rental Web Platform where users can browse products and add them to a cart for renting.
This project is built using basic HTML, CSS, and JavaScript and demonstrates frontend development concepts such as product cards, cart functionality, and local storage.

---

Live Website

Visit the live project here:
https://dhruwsimran81-git3.github.io/rentease-project/

---
 Features

- 🛏️ View rental products like Bed, Sofa, Table, Refrigerator, Washing Machine, and TV
- 🛒 Add products to cart
- ❌ Remove items from cart
- 💰 Total rent calculation
- 📦 Checkout functionality
- 🖼️ Product images displayed
- 📱 Mobile-friendly layout

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Browser LocalStorage
